# Relationship Coherence Patterns

Flexiformal patterns for the coherence of a *shared world between two people*, induced by close reading of a correspondence rather than specified top-down.

Where `stack-coherence/` keeps an artifact system coherent (evidence trails, blockers, sync), these keep coherent the mutually-built world of a relationship ‚Äî the "universe we find ourselves in" (Joe, 2026-04-12). The invariant is mutual intelligibility, attunement, and the freedom of that world to remain itself.

## Provenance-first

Unlike the stack families, these patterns are **inductive**: each was lifted from specific dated turns and then generalised, and each records its **instance of origin** in `@references-extra` (the move pioneered by `correspondence-coherence/mission-unlocks-eoi`). The corpus (`~/julia-chat/WhatsApp Chat with Julia Dallaway.txt`, 2026-02-27 ‚Äì 2026-06-29) doubles as the evidence landscape: the tag is the citation.

First induced set from a single corpus (the Joe/Julia correspondence). Treat as a candidate vocabulary to be widened, split, or pruned against further reading.

## Pattern Catalog

| Pattern | IF ‚Üí THEN (gist) |
|---|---|
| `quote-as-oblique-address` | truer than the moment can hold plainly ‚Üí send a found text **and gloss it in your own voice** |
| `lexicon-minting` | a moment/feeling recurs ‚Üí coin a handle (WoD, endearment) and curate the couple's canon |
| `low-pressure-bid` | you extend an invitation ‚Üí attach the permission-to-decline to the bid itself |
| `capacity-honoring` | one of you is depleted ‚Üí plan *around* the nervous system; rest is a move, not a withdrawal |
| `rupture-repair` | an emotion over-spills ‚Üí spiller names the cause; receiver de-personalises as shared journey |
| `narrative-suspension` | a label threatens to foreclose ‚Üí hold it open; incoherence over a distorting order |
| `parallel-lives-mirroring` | your situation rhymes with theirs ‚Üí name the symmetry (held open to correction) |
| `resonance-weaving` | a new input arrives ‚Üí weave it into the shared web rather than leave it loose (the coherence engine) |

The eight above were induced from the first ~3 weeks (an intellectual register: quotes, coinages, weaving, narrative-holding). A **second pass** over mid-April‚ÜíJune surfaced nine more ‚Äî the embodied / logistical / social register of shared life ‚Äî and a **third pass** over the final month surfaced one more (`affirmation-reframe`). Rather than a two-way *courtship / partnership* split (which over-claims a hard boundary and mis-files whole-relationship patterns like `rupture-repair`), the figures colour patterns by **vintage of introduction ‚Äî early / middle / recent** ‚Äî a purely descriptive, timestamp-derived banding: `early` = introduced by ~page 22 (the first ~3 weeks), `middle` = ~page 23‚Äì70 (mid-April‚Üílate May), `recent` = page ~71+ (late May‚ÜíJune). (Still no formal "phase" field on the turns: the corpus is time-linear, so ordering is legible from the timestamps.)

| Pattern (second pass) | IF ‚Üí THEN (gist) |
|---|---|
| `base-camp` | one is burdened/travelling ‚Üí offer your place as a shared hub with a stashed key |
| `meeting-the-people` | a friend/family/pet enters ‚Üí fold them into the circle, track approval/inclusion |
| `co-created-project` | an idea appears ‚Üí enrol it as a *named joint artifact* to build together |
| `held-uncertainty` | the future is unsettled ‚Üí hold it lightly and jointly, don't resolve it |
| `absence-as-spaciousness` | apart for a stretch ‚Üí read the distance as generative negative-space, not just loss |
| `running-gag` | a quirk recurs ‚Üí keep it as an affectionate in-joke that re-affirms the bond |
| `mutual-coaching` | one faces a work/outreach decision ‚Üí coach concretely, reciprocate, nudge (positive peer pressure) |
| `cook-as-care` | one is depleted/without food ‚Üí source/make/bring dinner; receiver flags the imbalance |
| `body-attunement` | the body flares (endo, cycle) ‚Üí name it plainly; attuned, non-fixing tenderness |

| Pattern (third pass) | IF ‚Üí THEN (gist) |
|---|---|
| `affirmation-reframe` | the world rejects/invites unfavourable comparison ‚Üí revalue the *person*'s worth and direction, don't just fix the task or endorse the verdict |

A **fourth pass** (a close re-reading of the two hardest emotional days) split a heavier sibling out of `rupture-repair`: `co-regulation`. Ordinary `rupture-repair` handles a single in-the-moment over-spill; `co-regulation` is for an *acute wobble in the relationship's felt security* (frazzled, "diminished", anxiety spilling "in a worrying way"), where held tension surfaces and is met with **space + co-regulation** (therapy/alone-time, a call, ü´Ç, meta-repair) rather than a quick de-personalising. It was previously hidden across four other tags (`rupture-repair`, `capacity-honoring`, `meeting-the-people`) until close reading unified it; those instances are now reclaimed.

| Pattern (fourth pass) | IF ‚Üí THEN (gist) |
|---|---|
| `co-regulation` | held tension about the relationship's felt security breaks as acute distress ‚Üí name it, give room to self-regulate (or receive the spill without personalising), co-regulate (call, ü´Ç, meta-repair); don't fix the feeling |

Each generative pattern names a failure twin (quote-flood, unilateral private-language, coerced bid, martyr-pushing, blame-of-the-critic, premature-labeling, projection, over/under-weaving, hollow-flattery) ‚Äî the anti-pattern side to be catalogued separately if useful.
