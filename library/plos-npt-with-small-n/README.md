# plos-npt-with-small-n

Presentational design patterns for drafting a small-n PLOS ONE-style qualitative or mixed-methods paper.

The seed corpus used here:

- 10.1371/journal.pone.0255564 - glaucoma clinical pathway, NPT, small-n multi-site implementation.
- 10.1371/journal.pone.0192224 - Medication Safety Thermometer, NPT-informed qualitative implementation.
- 10.1371/journal.pone.0195890 - ERAS surgery, NPT and coherence-heavy results structure.
- 10.1371/journal.pone.0282612 - stroke rehabilitation trial process work, NPT with focus groups and interviews.
- 10.1371/journal.pone.0239181 - AMBER care bundle, multi-method NPT process evaluation.
- 10.1371/journal.pone.0215873 - transgender primary care, multiple-case NPT study.
- 10.1371/journal.pone.0312943 - rehabilitation higher-education course-leader features, reflexive thematic analysis.
- 10.1371/journal.pone.0334692 - open science practices in HCI, interview study.
- 10.1371/journal.pone.0297969 - open research data sharing in epidemiology, mixed-methods phased design.
- 10.1371/journal.pone.0198606 - saturation and sample size in qualitative interviews.
- 10.1186/s12913-019-4398-1 - STORM/Choose Life cascade training, NPT; content cousin, not PLOS register model.
- 10.1186/s12913-021-06977-1 - DEALTS2 dementia train-the-trainer; content cousin, not PLOS register model.

Use these patterns as presentation constraints, not as content claims.  The repeated lesson across the PLOS exemplars is that small n is made credible by visible design: explicit empirical container in the abstract, tables that map sites/cases/data before findings, framework headings tethered to local empirical claims, and bounded transferability rather than generic generalisation.
